MySociety Application

A one stop application to manage the society, make and send bills, alot parking and aminities and make announcements.

Application Walk-through

The application has user login security system. After logining in the user will be displayed the respective homepage of member and adminstrator as per his/her login credentials. On the homepage the member can view and pay their bills, request for parking alotment, view announcements, etc. The adminstrator from his/her homepage can generate bills, make announcements, alot/dealot parking, etc. The interactive modern UI of the application helps the user to make the society operations smooth. The applciation maintains transperency and also makes repetitive work easy. This application will speed up the your society work.
