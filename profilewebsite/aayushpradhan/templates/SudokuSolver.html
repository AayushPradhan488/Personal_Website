Sudoku Solver

A human-like sudoku solver.

Procedure to use the Solver

The first step is to scan the cube, for this the cube is need to shown to the camera and is to be aligned with dots shown on the screen. The faces to be shown to the camera should be shown in specific orientation and in specific order only. After scanning the cube cross verify whether the program has rightly recognized the colours on each face or not. If there are any mistakes in colour recognition then we can manual edit the colour by mentioning the coordinate. Once the whole cube is correctly loaded into the program we need to start the algorithm to generate the solution. The algorithm will generate a solution based on "7 Step Algorithm" method. The solution is displayed in 3 parts, the first part marks the completion of first layer, the second part marks the completion of second layer and the third part marks the completionof third layer.
